1. HTML

HOME CSS JAVA

HTML, significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el lenguaje estándar usado para crear y estructurar el contenido de páginas web. HTML utiliza una serie de etiquetas para definir distintos elementos como encabezados, párrafos, imágenes, enlaces, tablas, formularios y otros tipos de contenido. HTML es la base de la web, y se utiliza junto con CSS (para estilos) y JavaScript (para interactividad) para crear páginas web completas y dinámicas.

2. CSS

HOME HTML JAVA

CSS, que significa Cascading Style Sheets (Hojas de Estilo en Cascada), es el lenguaje usado para definir el estilo y la presentación visual de documentos HTML. Mientras que HTML estructura el contenido de una página web, CSS controla cómo se ve dicho contenido, permitiendo aplicar colores, tamaños, fuentes, márgenes, posiciones y otros aspectos visuales de los elementos de una página. Especificidad y Cascada: CSS sigue reglas de especificidad para determinar qué estilos se aplican si hay varios que afectan al mismo elemento. La "cascada" significa que los estilos se aplican en un orden específico, donde las reglas más específicas (como los estilos en línea) predominan sobre las menos específicas.

3. JavaScript

HOME HTML CSS

JavaScript es un lenguaje de programación utilizado principalmente para agregar interactividad y dinamismo a las páginas web. Mientras que HTML se encarga de la estructura y CSS del diseño, JavaScript permite que los elementos de una página web respondan a las acciones del usuario, como clics, desplazamiento, entrada de texto, entre otras. Es fundamental para crear sitios web dinámicos y aplicaciones web interactivas. Lenguaje de programación interpretado: JavaScript es un lenguaje que se ejecuta directamente en el navegador, lo que significa que no necesita ser compilado antes de ser ejecutado. El navegador interpreta el código JavaScript cuando se carga la página web.
